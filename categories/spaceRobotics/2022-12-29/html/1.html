Space Mouse KeyBoard shortcuts #3Dprinting #3DThursday


PLA_parts shared this project on thingiverse!

Been using the space mouse for a bit but got a little frustrated moving from moving my hand from the space mouse to the keyboard. I looked online I saw people use a stream deck but it was very pricy so I found a cheaper keypad with a great interface.

I attached the fusion 360 file so you are welcome to edit the angle

the keypad that I am using is a Vaydeer, I found it on amazon, a very useful little keypad with a great interface and quality.

I designed the keycaps custom to my shortcuts

Download files: https://www.thingiverse.com/thing:5539313
